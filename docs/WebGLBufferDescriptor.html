    WebGLBufferDescriptor

    WebGLBufferDescriptor

    Data class for VAO buffer descriptions

    Constructor

    # new WebGLBufferDescriptor(location, buffer, size, type, offset, stride, normalize, divisor)

    Creates an object which represents a vertex attrib pointer.
    Parameters:
    Name Type Default Description
    location number The shader location this buffer needs to be bound to.
    buffer WebGLBuffer Buffer where the descriptor gets its values from
    size number The amount of components, can be 1 - 4.
    type GLenum A GLEnum describing the type used.
    offset number 0 The offset in the buffer; in bytes.
    stride number 0 Stride used for indexing the buffer; in bytes.
    normalize boolean false Defines if the values have to be normalized.
    divisor number 0 The divisor used for instanced drawing.
